# Estructura

Después de clonar el proyecto nos encontraremos con la siguiente estructura. El core del proyecto se encuentra dentro de la carpeta src.

.
├── src
│   ├── assets
│   │   ├── fonts
│   │   └── styles
│   │
│   ├── cfg
│   │   └── core.js
│   │
│   ├── components
│   │   ├── global
│   │   ├── Assistance.vue
│   │   └── ...
│   │
│   ├── functions
│   │   ├── fetchFunctions.js
│   │   └── formatDate.js
│   │
│   ├── App.vue
│   ├── event-bus.js
│   ├── main.js
│   └── routes.js
│
└── package.json

# assets

En la carpeta assets se encuentran todos los recursos de la aplicación como los archivos de estilos SASS ejemplo styles.scss, además de las typografias y font-icons.

Para los estilos, todos ellos convergen en el archivo styles.scss este archivo es el encargado de importar todos los demás archivos sass divididos en módulos para después ser utilizados por la aplicación.

# cfg

En la carpeta cfg se encuentra el archivo de configuración core.js que contiene la ruta de la API y el token de la aplicación.

const host = "http://URL:PORT/api/school";
const token = "186q2w3e4r5t6y9r2iecda5redboa02424312ac";

export { host, token };

# components

En la carpeta components se encuentran todos los componentes que conforman el proyecto, encontramos la carpeta global la cual contiene componentes generales que sirven a los demás.

# functions

En la carpeta funcitons se encuentra la función fetchFunctions.js que se encarga de hacer las peticiones a los diferentes servicios de la API utilizando el método nativo fetch.

También se encuentra la función helper formatDate.js que se encarga de formatear las fechas, ejemplo: 18 de junio del 2019

# App.vue

Componente principal del proyecto. En este componente es donde se importan todos los estilos del proyecto.

<style lang="scss">
@import "./assets/styles/styles.scss";
</style>

# event-bus.js

EventBus sirve para la comunicación entre componentes, permite emitir un evento en un componente y escuchar ese evento en otro.

// Emite
<script>
methods: {
  emitMethod () {
    EventBus.$emit('EVENT_NAME', payLoad);
  }
}
</script>

// Escucha
<script>
mounted() {
  EventBus.$on(‘EVENT_NAME’, function (payLoad) {
    ...
  });
}
</script>

# main.js

Archivo principal del proyecto, es aquí donde se importan las dependencias que harán funcionar todo el proyecto como vue, vue-router, etc. y la instancia principal de Vue.

# routes.js

Archivo donde se encuentran las diferentes rutas para las secciones del proyecto, en este archivo se enlazan los componentes a diferentes URLs.
